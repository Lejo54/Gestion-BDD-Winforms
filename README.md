## Conventions de nommage
Faut bien faire attention, on va tout nommer d'une certaine façon

**On abbrévie le composant en minuscule, et on rajoute le nom au bout**

*Exemples :*

* Label => lbl
* TextBox => txt
* Button => btn
* RadioButton => rdb
* CheckBox => chk 
* GroupBox => grp 
* Formulaire => frm 
* ListBox => lst 
* ComboBox => cbo 
* TreeView => trv 
* ErrorProvider => erp 
* Panel => pnl
* DataGridView => dgv
